# Приключения Василия

Простая анимационная игра на Pygame, где персонаж Василий проходит через различные сцены.

## Установка

1. Установите Python 3.7 или выше
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python vasily_adventure.py
```

## Управление

- **WASD** или **стрелки** - движение персонажа
- **ESC** - выход из игры

## Сцены

1. **Доставание меча из камня** - Василий находит меч в камне
2. **Преодоление препятствий** - Василий проходит через горы
3. **Бой с боссом** - Василий сражается с дерево-монстром
4. **Поиск ключей** - Василий собирает 3 ключа в пещере
5. **Открытие двери** - Василий открывает дверь и попадает на новый остров

## Особенности

- Автоматический переход между сценами каждые 5 секунд
- Интерактивные объекты (меч, ключи, дверь)
- Простая анимация персонажа
- Цветной интерфейс с информацией о прогрессе
